前缀和与差分，是比较简单而且基础的算法了，但它这个二维的我还没怎么看明白

前缀和简单来说就是前n项和，用一个数组来存原数组的前n项和，可以大大减少时间复杂度

这个其实前面已经用到过不少次了，我也已经比较熟练了，直接来道蓝桥杯真题吧

一维前缀和

k倍区间

题目描述

给定一个长度为 N 的数列，A1,A2,⋯AN，如果其中一段连续的子序列 Ai,Ai+1,⋯Aj ( i≤j ) 之和是 K 的倍数，我们就称这个区间 [i,j] 是 K 倍区间。

你能求出数列中总共有多少个 K 倍区间吗？

输入描述

第一行包含两个整数 N 和 K( 1≤N,K≤1e5 )。

以下 N 行每行包含一个整数 Ai ( 1≤Ai≤1e5 )

输出描述

输出一个整数，代表 K 倍区间的数目。

这个找多少个区间的有点难理解，需要知道一个数学定理

如果两个数对一个数取余得到的余数相同，那么这两个数的差就能整除那个数

再来道蓝桥杯真题

统计子矩阵

问题描述

给定一个 N×M 的矩阵 A, 请你统计有多少个子矩阵 (最小 1×1, 最大 N×M) 满足子矩阵中所有数的和不超过给定的整数 K ?

输入格式

第一行包含三个整数 N,M 和 K.

之后 N 行每行包含 M 个整数, 代表矩阵 A.

输出格式

一个整数代表答案。

评测用例规模与约定

对于 30% 的数据, N,M≤20.

对于 70% 的数据, N,M≤100.

对于 100% 的数据, 1≤N,M≤500;0≤Aij≤1000;1≤K≤250000000.

除了前缀和以外，还用到了一个滑动窗口，固定左右边界后滑动上下边界
