开始学习单片机，社团里正好有个蓝桥杯单片机比赛用的51开发板，就先用它来学。

今天学了怎么点亮LED灯，这块开发板和其他的相比，点灯有点复杂，因为它芯片的io口没有直接和LED灯相连，而是经过了一个锁存器，所以有一点点麻烦。

第一步

使用开发板的第一步是看原理图，要点亮LED就先看和LED有关的原理图

这是这块板子上LED部分的原理图，可以看到右侧输入默认是VCC，也就是高电平，所以如果想点亮它，要将左侧输入改为低电平，导通电流，LED灯就会被点亮。

左侧的HC573是一个锁存器，当Y4C输入为1(也就是高电平)时，D端和Q端联通，D端为1时Q端也为1，当Y4C输入为0时，Q端不受D端影响。

所以我们的思路应该是首先将Y4C置为1，再去改变P0的输入。

在原理图中可以看到，Y4C的左侧连着一个或非门，或非门是先取或再取非，所以只有当Y4和WR输入都为0（即低电平）时Y4C才会为1，IO模式下WR默认为0，所以要将Y4也置为0。

而要将Y4置为0又要看Y4连接着什么，是不是挺绕的。Y4由上面这个HC138控制，它的输出可以直接去查真值表，而且也很好记，ABC三个输入能改变Y0到Y7八个输出，将其中一个置为低电平，而其余都为高电平，记忆方法是将C B A看成二进制的三个位，C B A为0 0 0时Y0为低电平，C B A为0 0 1时Y1为低电平，0 1 0时Y2为低电平，也就是说如果想要把Y4置为低电平，只要将C B A改为 1 0 0就可以了。

想实现依次点亮，我们可以通过对0xff进行位操作来实现

在c语言中， << 符号代表左移一位，0xff(1111 1111)左移一位的话就变成了0xfe(1111 1110)，这样就实现了点亮其中一个LED灯，左移两位就变成了 1111 1100。我们只要让它左移8次，每次多移动一位就可以实现依次点亮的效果了，可以用一个for循环轻松实现
